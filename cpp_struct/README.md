### 结构体

**1. 概念**

- 用户自定义数据类型，允许用户存储不同的数据类型

**2. 结构体定义和使用**

- 语法： 

  ```c++
  struct 结构体名 {结构体成员列表};
  ```

- 创建结构体变量三种方法（struct关键字可省略）

```c++
struct 结构体名 变量; // 通过 . 访问变量中的属性
struct 结构体名 变量 = { 成员初始化 }; 
// 定义结构体时即创建变量
```

**3. 结构体指针**

- 利用操作符 -> 可以通过结构体指针访问结构体属性

**4. 结构体做函数参数**

- 值传递
- 地址传递
